# 健康紀錄管理系統 (Health Management System)

本專案是一個具備風險評估邏輯的健康紀錄管理系統，採用前後端分離架構，實現了完整的 CRUD 操作，並整合了基於決策樹的多層次風險判定邏輯。

## 🚀 系統特色

* **多層決策樹演算法**：根據睡眠時數、當日步數及心情分數，進行三層式的健康風險等級判定（高風險/中風險/低風險）。
* **資料完整性**：內建輸入驗證機制，防止非法數值（如睡眠時數超出 0-24 小時）寫入資料庫。
* **互動體驗**：具備視覺回饋與回到頂部動畫效果。
* **資料庫管理**：支援完整的增刪改查 (CRUD) 與資料清空功能，確保資料管理的靈活性。

## 🛠️ 技術棧

* **Frontend**: HTML5, CSS3, JavaScript (Fetch API)
* **Backend**: Node.js, Express.js
* **Database**: SQLite3
